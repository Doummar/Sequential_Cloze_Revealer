# Sequential Cloze Revealer

A focused and distraction-free way to review cloze cards in Anki.

Sequential Cloze Revealer allows information to be revealed progressively instead of exposing everything at once, helping you focus on one idea at a time and making active recall more effective.

Originally built for my own daily study sessions and later shared with the community.

---

## Why I Made It

While studying, I often found large cloze cards visually overwhelming.

Sometimes I only wanted to reveal one part of the answer while keeping everything else hidden.

Sequential Cloze Revealer was created to solve that problem by introducing a cleaner and more focused review experience.

Today it remains one of the add-ons I use most during my own studies.

## How to Use

1. Open a card containing supported clozes.
2. Click hidden sections to reveal them.
3. Press Space or Enter to continue reveal sequences.
4. Use shortcuts to display additional information when needed.
5. Rate the card as normal.

### Manual Installation

1. Download this repository.
2. Extract the files.
3. Move the add-on folder into:

#### Windows

```text
C:\Users\YOUR_NAME\AppData\Roaming\Anki2\addons21\
```

#### macOS

```text
~/Library/Application Support/Anki2/addons21/
```

#### Linux

```text
~/.local/share/Anki2/addons21/
```

4. Restart Anki.
